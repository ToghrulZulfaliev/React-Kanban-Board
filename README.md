A modern Kanban task board built with **React** and **Redux Toolkit**, featuring **drag & drop**, **search**, **dark mode**, and **LocalStorage persistence**.

 **Features**
- Kanban columns: **To Do / Doing / Done**
- **Drag & Drop** tasks between columns (@dnd-kit)
- **Add / Edit / Delete** tasks (modal forms)
- **Search** tasks (title/description/tags)
- **Dark / Light mode**
- **LocalStorage** persistence (tasks saved after refresh)
- Responsive layout (Tailwind CSS)

** Tech Stack**
- React (Hooks)
- Redux Toolkit
- @dnd-kit
- Tailwind CSS
- Formik + Yup
- Vite
